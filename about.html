About the Website

This project is a showcase of vintage cameras, featuring interactive 3D models** created in Blender, high-quality renders, and detailed descriptions. The goal is to provide an immersive experience, allowing users to explore the beauty of classic cameras.

3D Models (Created in Blender)

All 3D models were meticulously crafted in Blender, with attention to detail in modeling, texturing, and rendering.

A high-poly model of a Fujica Compact camera, textured in Substance Painter.

This classic Kodak film camera was modeled and UV-mapped in Blender, then textured using procedural shaders.

Design & Integration

The website is built using HTML, CSS, JavaScript, and Bootstrap, ensuring a modern and responsive layout. Three.js is used for rendering interactive 3D models, and Bootstrap provides a clean structure.

  • ✅ Modeling: Created in Blender, optimized for web display.
  • ✅ Texturing: Substance Painter & procedural shaders.
  • ✅ Web Integration: Three.js for 3D models, Bootstrap for UI.
  • ✅ Interactivity: Lightbox gallery, smooth animations.
